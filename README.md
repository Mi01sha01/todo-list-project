# TODO-LIST PROJECT
A simple and attractive Todo List web app built using HTML, CSS and JavaScript.
It helps you manage your tasks with ease.

# FEATURES
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Responsive
- Local storage support

# Technologies Used
- HTML5
- CSS3
- JavaScript
